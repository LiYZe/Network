# UDP协议

由"标头"和"数据"两部分组成。

"标头"主要定义了发出端口和接收端口，8个字节，总长度不超过65,535字节。

"数据"部分就是具体的内容。

优点：简单，容易实现

缺点：可靠性较差

# TCP协议

优点：确保不会遗失

内嵌在IP数据包的"数据"部分。

TCP数据包没有长度限制
